flex요소

display css로 레벨 변경이 가능하다. flex요소가 될경우 inline, block, inline 레벨을 벗어난 별도의 레이아웃이 된다.

flex선언시 부모는 flex 컨테이너. 자식요소는 flex item으로 동작한다. 이후 모든 개체는 flex 레이아웃 규칙을 따른다.

Flex 컨테이너 관련 속성

flex-direction : 자식 요소(flex item)의 정렬 방향.

flex-wrap :아이템을 한 줄에 다 넣을지 줄바꿈할지 결정.

flex-flow : flex-direction + flex-wrap을 한 번에 설정.

justify-content : 주축(가로 or 세로) 방향 정렬.

align-items : 교차축(세로 or 가로) 방향 정렬.

align-content : 여러 줄(flex-wrap 사용 시) 간격 정렬.

Flex 아이템 관련 속성

order : 요소의 순서 변경.

flex-grow : 남는 공간을 얼마나 차지할지 (비율).

flex-shrink : 공간 부족 시 얼마나 줄어들지 (비율).

flex-basis : 요소의 기본 크기.

flex : flex-grow, flex-shrink, flex-basis를 한번에 설정.

align-self : 해당 아이템만 개별적으로 교차축 정렬.
